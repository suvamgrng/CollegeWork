Aspect ✓ CHECKED ⚡ UNCHECKED
Must Handle? ✅ YES - Compiler enforces ❌ NO - Optional
When Detected? Compile-time (before running) Runtime (during execution)

Extends Exception (directly) RuntimeException (extends Exception)

Code Examples

1️⃣ Basic Try-Catch-Finally

try { // Code that might throw exception int result = 10 / 0; // ArithmeticException } catch (ArithmeticException e) { // Handle the exception System.out.println("Cannot divide by zero!"); e.printStackTrace(); } finally { System.out.println("Cleanup code here"); }

2️⃣ Multiple Catch Blocks

try { int[] arr = {1, 2, 3}; int result = Integer.parseInt("abc"); int value = arr[10]; } catch (NumberFormatException e) { System.out.println("Invalid number format"); } catch (ArrayIndexOutOfBoundsException e) { System.out.println("Array index out of bounds"); } catch (Exception e) { System.out.println("General exception"); }

3️⃣ Try-With-Resources (Auto-Close)

try (FileReader file = new FileReader("file.txt")) { int character = file.read(); System.out.println((char) character); } catch (IOException e) { System.out.println("Error reading file: " + e.getMessage()); } // FileReader is automatically closed here

4️⃣ Throwing Exceptions

public void validateAge(int age) throws IllegalArgumentException { if (age < 0 || age > 150) { throw new IllegalArgumentException("Invalid age"); } System.out.println("Valid age: " + age); } // Using it: try { validateAge(-5); } catch (IllegalArgumentException e) { System.out.println("Caught: " + e.getMessage()); }

5️⃣ Custom Exception

class InvalidAgeException extends Exception { public InvalidAgeException(String message) { super(message); } } try { throw new InvalidAgeException("Must be 18+"); } catch (InvalidAgeException e) { System.out.println("Custom: " + e.getMessage()); }
